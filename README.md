# Space & Time — Ārṣa Cosmology and Chronometry

Concise reference combining two companion documents: a detailed Ārṣa cosmology ([space.md](space.md)) and a complete synthesis of Vedic temporal metrology ([time.md](time.md)). Useful for students, translators, and researchers of classical Indian cosmology and calendrics.

## Files

- [space.md](space.md) — Brahmāṇḍa Darśana: cosmography, Lokā-architecture, cosmic oceans, and Purāṇic sources.
- [time.md](time.md) — Ārṣa chronometry: Bhāgavata atomic ladder, Manu/Viṣṇu chains, Siddhāntic notes, and concordances.

## Usage

Open the [space.md](space.md) and [time.md](time.md) files in this repository to read the full texts and references. Use the GitHub web UI or any Markdown viewer.
